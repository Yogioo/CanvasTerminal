# MSDF Atlas for M3 MVP

Generated by `msdf-atlas-gen` v1.4.0.

## Command

```cmd
msdf-atlas-gen.exe ^
  -font "C:/Windows/Fonts/NotoSansSC-VF.ttf" ^
  -type mtsdf ^
  -format png ^
  -json assets/fonts/msdf/atlas.json ^
  -imageout assets/fonts/msdf/atlas.png ^
  -size 48 ^
  -pxrange 4 ^
  -charset assets/fonts/msdf/charset.txt ^
  -fontname "NotoSansSC-VF"
```

## Files

| File | Size | Description |
|------|------|-------------|
| `atlas.png` | 123 KB (344×344) | MTSDF atlas image (RGBA). RGB = MSDF, A = SDF. |
| `atlas.json` | 24 KB | Glyph layout metadata. |
| `charset.txt` | 83 B | Charset spec: ASCII printable + 9 Chinese chars. |

## Notes

- Texture format: `Rgba8Unorm` (NOT sRGB – MSDF data is linear).
- GPU rendering via egui-wgpu PaintCallback with WGSL MSDF shader.
- y-origin = bottom (default). UV Y flipped for wgpu texture coordinates.
